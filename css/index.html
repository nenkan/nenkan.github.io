Spacing

The default spacing should be `8px`. If you want to decrease the spacing, keep dividing by 2 until you are satisfied: `4px`, `2px`, `1px`. If you want to increase the spacing, keep multiplying by 2 until you are satisfied: `16px`, `32px`, `64px`. For more detailed spacing: `24px`, `48px`.
